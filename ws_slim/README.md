## To start a jupyter lab:


1. Go to the directory of this workshop: `cd ws_slim/`
1. Install the environment for this workshop: `pixi install`
1. Clone the repository given by the presenter
1. Start a Jupyter lab: `pixi run jupyter lab` leave the terminal open. The Jupyter lab will open in a tab of your default browser
1. If you wish to use a different browser, keep the previous terminal open and copy the address: http://localhost:8888/lab
